## 📖 About the Project

CESS (Civil Engineering Student Society) is a non-profit student organization committed to technical excellence, innovation, and the holistic development of civil engineers at NIT Arunachal Pradesh.

This repository contains the source code for the official CESS website. It serves as a unified digital platform for students, faculty, and alumni to stay updated on:

* Technical events
* Field exposures
* Competitions (Ideathons, Bridge Making, etc.)
* Leadership messages

---

## ✨ Key Features

* 🎨 **Modern Pastel UI:** Custom earthy palette (Mint, Peach, Ice Blue, Pistachio) for a clean academic look
* 📱 **Fully Responsive:** Works smoothly on desktop, tablet, and mobile
* 🖼️ **Interactive Gallery:** JavaScript-based image slider for event showcases
* 🎬 **Scroll Animations:** Smooth animations using Intersection Observer API
* ⚡ **Lightweight & Fast:** No heavy frameworks, optimized performance

---

## 🛠️ Tech Stack

* HTML5 (Semantic structure)
* CSS3 (Flexbox, Grid, Animations, Variables)
* Vanilla JavaScript (DOM, Events, Slider Logic)

## 🚀 How to Clone and Run Locally

Follow these steps carefully:

### 📌 Prerequisites

* Install Git → https://git-scm.com/
* Verify installation:

```bash
git --version
```

---

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/arunkumar731/civil-engineering-student-society-nit-ap-website.git
```

---

### 2️⃣ Navigate to Project Folder

```bash
cd civil-engineering-student-society-nit-ap-website
```

---

### 3️⃣ Run the Website

### ✅ Method A: Open Directly in Browser

* Open the project folder
* Double-click `index.html`

---

### ✅ Method B: Using VS Code (Recommended)

* Open project in VS Code
* Install **Live Server** extension
* Right-click `index.html` → **Open with Live Server**
